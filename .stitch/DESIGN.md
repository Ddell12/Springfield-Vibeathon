## 1. Overview & Creative North Star

**Creative North Star: "The Digital Sanctuary"**

This design system bridges the gap between clinical reliability and emotional warmth. We move away from the "cold dashboard" aesthetic toward a "Sanctuary" layout тАФ one that feels like a quiet, well-organized room.

The system breaks traditional SaaS "template" looks by rejecting rigid borders in favor of **Tonal Enclosure**. We use intentional asymmetry and high-contrast typography scales (pairing the geometric warmth of Manrope with the functional clarity of Inter) to create a rhythm that feels human and curated. It is professional enough for a therapist, yet soft enough for a child to navigate without feeling patronized.

### The "No-Line" Rule

**Standard 1px borders are strictly prohibited for sectioning.**
Visual boundaries must be defined exclusively through background color shifts. A section should be distinguished by placing a `surface-container-low` (#f0f3ff) element against a `surface` (#f9f9ff) background. This creates "soft edges" that reduce visual anxiety.

## 4. Elevation & Depth

Depth is organic, not mechanical.

- **Layering Principle:** Stack `surface-container-highest` inside `surface-container-low` to create "recessed" wells for input fields or data lists. No shadows needed.
- **Ambient Shadows:** For floating elements (FAB, Tooltip): `0px 12px 32px rgba(21, 28, 39, 0.06)`. Shadow color is a tint of `on-surface`, never pure black.
- **"Ghost Border" Fallback:** For accessibility, use `outline-variant` (#bec9c9) at **15% opacity**. A 100% opaque border is a design failure.

## 7. Do's and Don'ts

### Do

- Use asymmetrical margins in hero sections for editorial feel
- Use `tertiary-fixed` (#ffdbca) for warm highlights ("Notes from Therapist")
- Use `3.5rem` (spacing 10) between sections for breathing room
- Use `sm` (4px) minimum radius on all elements

### Don't

- Use 100% black text тАФ always `on-surface` (#151c27) or `on-surface-variant` (#3e4949)
- Use sharp 0px corners anywhere
- Use animations faster than 300ms тАФ all transitions use `cubic-bezier(0.4, 0, 0.2, 1)`
- Use standard 1px borders for sectioning (use tonal shifts instead)
