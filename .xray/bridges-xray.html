Purpose and Boundaries
The builder slice is the core value proposition of Bridges. It owns the streaming SSE code generation pipeline: user describes a therapy tool in plain language, Claude streams React code with tool calls (write_file, generate_image, generate_speech), files are bundled server-side via Parcel into a self-contained HTML, and rendered in a sandboxed iframe. Does NOT own: publishing (sharing slice), user auth, or template browsing.
